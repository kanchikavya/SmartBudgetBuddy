# 💰 Smart Budget Buddy

A simple and interactive **Expense Tracker** built using **HTML, CSS, and JavaScript**. This application helps users manage their daily expenses by adding, categorizing, and tracking spending in an easy-to-use interface.
## 📌 Project Overview
Smart Budget Buddy is a lightweight web application that allows users to:
- Add daily expenses
- Categorize expenses
- Filter expenses by category
- View the total amount spent
- Visualize spending using charts
- Toggle between Light Mode and Dark Mode
The application stores data in the browser using **Local Storage**, so expenses remain available even after refreshing the page.

## ⚙️ How to Run the Project
### Step 1
Clone the repository
```bash
git clone https://github.com/kanchikavya/SmartBudgetBuddy.git
```
### Step 2
Open the project folder
```bash
cd SmartBudgetBuddy
```
### Step 3
Open `index.html` in any web browser.
No additional installation is required.
## 📖 How to Use
1. Enter the Expense Title.
2. Enter the Expense Amount.
3. Select the Date.
4. Choose a Category.
5. Click **Add Expense**.
6. View the updated expense list.
7. Filter expenses using the dropdown.
8. View the total expense.
